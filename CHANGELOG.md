# 版本变更日志

## v1.2.0 (2026-03-28 23:54)

**正式功能版**：多项体验升级 + 运营洞察 + 稳定性修复

### 新增
- 💡 **用户反馈系统**：实时看板顶部导航新增"💡 反馈"入口，支持提交需求/BUG/建议；管理后台新增反馈管理卡片，支持状态流转、筛选和一键复制，便于开发排期
- 🔴 **正在直播自动开卡片**：打开看板时，自动监控列表中正在直播的账号自动启动并展示卡片
- 👁 **分组过滤器补全**：看板新增「⚔️ 竞品」分组筛选 Tab
- 🔥 **话术节点摘要卡**：流量涨跌时刻的主播话术自动提炼，独立高亮展示
- 🌐 **话术时间轴默认中文**：气泡直接显示中文翻译，非中文附带原文标注
- 📱 **全站响应式适配**：session_detail / rivals / automonitor / compare / history 均支持 640px/480px 断点
- ➕ **自营对比页账号管理**：支持新增账号和删除商户
- 🌟 **复盘话术高频词气泡图**：按词频分层着色，直观呈现关键词
- 🤖 **AI 总结免费方案**（Pollinations.ai）：无需 API Key 即可使用，优先 Gemini 降级免费 AI
- 🔗 **连接状态 Badge**：已连接/直播中 红色醒目状态指示
- 🃏 **卡片拖拽排序**：原生 HTML5 Drag&Drop，拖动调整监控卡片顺序
- 📡 **卡片实时 AI 摘要**：连接后自动 30s 首次更新，每 2 分钟轮播
- 👑 **用户管理摘要**：管理后台展示每用户场次/账号数/最后活跃时间

### 优化
- 🕐 **开播时间显示北京时间**：修复时区解析问题
- ⏱ **已播时长统计**：从系统监测到账号开播起计时
- ❤️ **卡片第4格改为点赞数**：替代礼物收入，实时更新
- 🎙️ **话术摘要更快触发**：第1条话术即更新摘要；AI 摘要提前至连接后10秒
- ⚡ **评论翻译提速 50-70%**：持久 httpx 连接池，极短文本跳过翻译
- 🌍 **语言分析精确化**：其他语种独立显示具体国旗+语种名（21 种）
- 🗺️ **方言国旗精确化**：阿语四大方言各自配国旗 emoji
- 🎙️ **语音采集稳定性升级**：指数退避重连，MAX_CONSECUTIVE_ERRORS=8
- 🔊 **阿语 Whisper 识别优化**：连续段 hint_lang='ar'，幻觉过滤
- 🧠 **语言识别升级 v3**：langid + langdetect 双层降级，英文短句词典法，准确率 100%
- 🔧 场次/时长列独立展示（compare 对比页）
- 🔧 峰值在线进度条 UI 修复（不再溢出单元格）
- 🔧 自营对比页同一账号合并为一行
- 🔧 直播状态判断修复：`status=live && !end_time` 双重判断防误报

### Bug 修复
- 修复复盘页所有数据区域一直显示"加载中..."（`</script>` 标签缺失导致 JS 完全不执行）
- 修复评论摘要始终显示"等待数据..."（comment-text 类名缺失）
- 修复开播时间时区问题导致已播时长算错
- 修复多页面历史场次显示"直播中"（僵尸 session 前端防御）
- 修复 emoji 正则 `\u1F300` 书写错误（英文评论无法翻译）
- 修复 statusData ReferenceError（const 作用域问题）
- 修复 `_startTs` → `startTs` 计时器不工作
- 修复重复 LANG_META 定义冲突
- 修复 /api/session/<id>/review 路由装饰器缺失（永久 404）
- 修复 api_rivals_recommend 中空 own_users 的 SQL 语法错误
- 修复 speechCount 路径错误、requestAiSummary event 参数缺失

### 清理
- 🗑️ 删除实时看板竞品账号管理卡片（相关 CSS 约25行 + JS 函数约55行）
- 🗑️ 删除导航栏「📱 远程访问」按钮及相关弹窗（已有固定网址方案）

## v1.0.0 (2026-03-26 22:46)

**初始正式版本**

- 语音采集（Whisper large-v3）+ 主播话术中文翻译

- 竞品分析 + 智能推荐
- 评论/话术翻译（Google Translate，走本地代理）
- 微信推送（WxPusher）
- ffmpeg 拉流代理支持（HLS 走 HTTP 代理）
- 复盘加载优化（评论翻译改为存库+按需读取，加载时间 78s→277ms）
