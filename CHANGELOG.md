# 版本变更日志

## v1.1.0 (2026-03-27)

**功能迭代版本**

### 新增
- 📊 自营对比分析页（`/compare`）：账号横向排名 / 趋势图 / 自营vs竞品雷达图

- 卡片实时显示评论/话术高频词动态小总结

- 话术时间轴与趋势图 X 轴精准对齐
- 历史页支持 `?u=账号名` URL 参数自动填充搜索
- 统一各页面导航栏样式，新增「📊 自营对比」入口

## v1.0.0 (2026-03-26)
**初始正式版本**

### 功能清单
- 实时监控 5-10 个 TikTok 账号并发直播
- 4 列并排看板，实时显示在线人数/评论/礼物
- 语音采集（Whisper large-v3）+ 主播话术中文翻译
- 多语言检测：中/英/阿拉伯语 + 四大方言（海湾/埃及/黎凡特/北非）
- 方言雷达面板（看板+复盘双层面板）
- 历史复盘：指标对比/趋势图/话术时间轴
- 多用户登录系统（登录/注册/权限分级）
- 自动监控列表（启动自动恢复/批量导入）
- 竞品分析 + 智能推荐
- 评论/话术翻译（Google Translate，走本地代理）
- 微信推送（WxPusher）
- 远程访问（Cloudflare Tunnel）
- ffmpeg 拉流代理支持（HLS 走 HTTP 代理）
- 复盘加载优化（评论翻译改为存库+按需读取，加载时间 78s→277ms）
