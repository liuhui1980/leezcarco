# 版本变更日志

## v1.3.3 (2026-04-01)

**监控逻辑修正版**：修复 watch 账号采集行为、分组标签数据源、UI 文案准确性

### 修复
- 🐛 **watch 账号恢复轻量监控设计**：关注账号不再进行全量数据采集（评论/礼物/话术/翻译），仅保留开播检测 + 微信推送通知功能，同时创建轻量 session 记录用于开播追踪
- 🐛 **分组标签数据源迁移**：历史记录页和实时看板的账号分组标签从 localStorage 迁移到后端 API（`/api/account/groups`），消除浏览器缓存清除导致分组标签丢失的脆弱依赖
- 🐛 **自动监控页文案修正**：底部提示"竞品账号：轻量采集（仅在线人数）"更正为"竞品账号：全量采集（与自营能力一致）"，与 v1.3 统一采集行为一致

### 新增
- 📡 **新增 `/api/account/groups` 接口**：返回当前用户的所有账号分组映射 `{username: group_name}`，作为前端分组标签的唯一数据源
