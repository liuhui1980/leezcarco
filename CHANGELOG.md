# 版本变更日志

## v1.1.1 (2026-03-28)

**全面优化版本**：UI 修复、响应式适配、话术时间轴增强、竞品面板清理

- 🌐 话术时间轴默认显示中文翻译：气泡直接显示中文，非中文时附带原文小标注
- 📱 全站响应式优化：session_detail / rivals / automonitor / compare / history 均支持 640px / 480px 断点
- ➕ 自营账号对比页支持新增账号和删除商户功能

### 优化

- 🔧 话术时间轴绝对定位模式中，峰值节点 z-index 置顶，奇偶错行防遮挡加强
- 🔧 直播状态判断修复：`status=live && !end_time` 双重判断防误报

## v1.1.0 (2026-03-27)（合并至 v1.1.1）

**体验增强版本**（12项功能）

### 新增
- 🌟 复盘话术高频词气泡图：按词频分层着色，直观呈现关键词
- 🤖 AI 总结免费方案（Pollinations.ai）：无需 API Key 即可使用，优先 Gemini 降级免费 AI
- 🔗 连接状态 Badge：已连接/直播中 红色醒目状态指示
- 🃏 卡片拖拽排序：原生 HTML5 Drag&Drop，拖动调整监控卡片顺序
- 📡 卡片实时 AI 摘要：连接后自动 30s 首次更新，每 2 分钟轮播新总结
- 👑 用户管理使用摘要：管理后台展示每个用户的场次/账号数/最后活跃时间

### 优化
- ⚡ 评论翻译提速 50-70%：持久 httpx 连接池，极短文本跳过翻译
- 🌍 语言分析一致性修复：其他语种独立显示具体国旗+语种名（21 种）
- 🗺️ 方言国旗精确化：阿语四大方言各自配国旗 emoji
- 🎙️ 语音采集稳定性升级：指数退避重连，MAX_CONSECUTIVE_ERRORS=8
- 🔊 阿语 Whisper 识别优化：连续段 hint_lang='ar'，幻觉过滤
- 🧠 语言识别模型升级 v3：langid + langdetect 双层降级，英文短句词典法，准确率 100%

### Bug 修复
- 修复 emoji 正则 `\u1F300` 书写错误导致英文字母被过滤（英文评论无法翻译）
- 修复 statusData ReferenceError（const 作用域问题）
- 修复 `_startTs` → `startTs` 计时器不工作
- 修复重复 LANG_META 定义冲突
- 修复 /api/session/<id>/review 路由装饰器缺失（永久 404）
- 修复 api_rivals_recommend 中空 own_users 的 SQL 语法错误
- 修复 speechCount 路径错误、requestAiSummary event 参数缺失等
